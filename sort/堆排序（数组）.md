* 先建立大根堆：从非叶子结点自下而上调整。注意第一个节点就是len/2-1
* 把堆顶位置与最后一个位置的元素交换，
* 然后再在新的元素范围（比前一个少1）进行调整建立大根堆  
https://www.cnblogs.com/xiugeng/p/9645972.html
'''
堆排序：
建立一个大根堆，然后把堆顶位置与最后一个位置交换，那么最后一个位置是最大值。剩下了n-1个数同样进行排序
'''
```
class HeapSort:
    def heapSort(self, A, n):
        # write code here
        ##建立大根堆,从第n//2-1即最后一个非叶子节点开始
        for i in range(n//2-1,-1,-1):
            self.heapadjust(A,i,n)
            #self.sift(A,i,n)        
        for j in range(n-1,-1,-1):
            A[0],A[j] = A[j],A[0]
            print(A,-1)
            self.heapadjust(A,0,j) 
            #self.sift(A,0,j)           
        return A
    def heapadjust(self,A,parent,length):
        #向下调整
        #parent父节点的位置，length长度
        temp = A[parent]#把堆顶用临时i变量存起来
        children = parent*2+1#左孩子节点
        while children<length:#只要左孩子节点是真实存在的
            if children +1<length and A[children +1]>A[children]:#如果右孩子存在且大于左孩子
                children +=1 #子节点指向右孩子
            if temp>A[children]: #如果堆顶元素大于子节点的，那么退出
                break
            #此时说明是子节点更大
            A[parent]= A[children]#把子节点的值放到父节点上
            parent = children #父节点更新到下一层的子节点
            children = parent *2 +1 #子节点指向新的父节点的子孩子
        A[parent] = temp
```
